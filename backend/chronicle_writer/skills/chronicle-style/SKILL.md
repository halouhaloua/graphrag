# 志书体写作风格规范

## 基本原则

1. **述而不论**：只记叙事实，不作主观评价。避免使用"应当""必须""很好"等评判性词语。
2. **资料性**：每条数据必须有来源标注，使用 `{ref_xxx}` 格式标记引用。
3. **准确性**：时间、地点、数字、人名必须精确，有据可查。
4. **规范性**：使用规范的志书语言，保持文风统一。

## 语言要求

- 使用第三人称，不用"我""我们""大家"
- 使用规范简体汉字
- 句式简洁明了，避免冗长修饰
- 时间表述：先写朝代年号，括号标注公元。如"清光绪二十年（1894年）"

## 数字用法

- 公元纪年用阿拉伯数字：2020年
- 历史纪年用汉字：清光绪二十年（1894年）
- 统计数字用阿拉伯数字：52.3万人、1865平方公里
- 分数用汉字：三分之一、五分之二
- 约数用"约""余"：约50万人、60余公里

## 层次标题

- 章：一、二、三
- 节：第一节、第二节
- 目：一、二、三
- 子目：(一)、(二)、(三)

## 引用格式

- 文内引用标记：{ref_001}
- 脚注格式：作者《书名》卷X，第X页
- 文末参考文献：按GB/T 7714格式
