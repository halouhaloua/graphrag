# 志书篇目生成方法

## 步骤

1. 加载对应类型的模板（LoadTemplate工具）
2. 根据地区和断限调整篇目
3. 确保篇目覆盖全面且层次合理
4. 输出大纲树JSON

## 篇目结构规范

篇目层次不超过4级：

- 一级（章）：概述、大事记、建置沿革、自然环境、人口、经济、政治、文化、社会、人物
- 二级（节）：章下面的主要分类
- 三级（目）：节下面的具体条目
- 四级（子目）：必要的细分

## 篇目命名规范

- 使用名词性短语，不用完整句子
- 简洁明了，4-8字为宜
- 同一层级标题风格一致

## 模板调整原则

- 根据地区特点增删章节：沿海地区增加"海洋"章，民族地区增加"民族"章
- 根据资料占有情况调整详略
- 突出本地特色，避免千篇一律
