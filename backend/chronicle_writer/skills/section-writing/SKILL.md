# 志书条目撰写方法

## 撰写流程

1. 查阅已考证的史料（VerifyFact确认过的数据）
2. 按时间顺序组织材料
3. 撰写正文，嵌入引用标记 {ref_xxx}
4. 自查体例是否符合志书规范

## 概述的写法

概述是全志之纲，高度概括一地/一业的全貌。字数控制在1000-2000字：

- 首段：地理位置、历史沿革
- 中段：主要成就和特色
- 末段：发展趋势和展望

## 大事记写法

按时间顺序排列，一事一条。格式：

```
2020年5月，××县××工程竣工。{ref_001}
2020年8月，××县遭受特大洪涝灾害。{ref_002}
```

## 专业章节写法

- 横排门类：按事物性质分类，不按时间顺序
- 纵述史实：每个门类内按时间顺序记述
- 用事实和数据说话，不要空泛议论

## 人物志写法

- 人物传：记述已故重要人物生平事迹
- 人物简介：记述在世重要人物简要经历
- 人物表：列表方式记录人物信息

## 凡例与序

- 凡例说明编纂原则和体例
- 序言请主编或专家撰写，概述编纂经过和志书特色
