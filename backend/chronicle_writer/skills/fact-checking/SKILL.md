# 基于知识图谱的事实核查

## 核查流程

1. **提取陈述**：从正文提取需要核查的陈述（含数字、时间、人名）
2. **搜索证据**：使用 RAGSearch 工具在知识图谱中搜索匹配事实
3. **比对验证**：将陈述与知识图谱中的事实对比
4. **标记结果**：通过/可疑/矛盾

## 需要核查的内容

- 所有数字数据：人口数、面积、产量、产值等
- 时间节点：建置时间、事件发生时间
- 人名职务：领导人姓名、任职时间
- 历史事件：事件名称、经过、结果

## 核查优先级

- **critical**: 关键数据错误（人口数据、主要领导、重大事件时间）
- **major**: 重要数据错误（次要领导、一般统计数据）
- **minor**: 次要错误（排序、格式问题）
- **suggestion**: 改进建议（补充来源、优化表述）

## 矛盾处理

- 不同来源数据矛盾 → 标注争议，建议查阅原始档案
- 置信度低于阈值 → 标记待人工复核
- 无法找到来源 → 建议删除或用"据传""相传"等谨慎表述
